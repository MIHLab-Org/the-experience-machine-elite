# The Experience Machine: Elite Dangerous

An LLM-powered philosopher-persona NPC conversation prototype for Elite Dangerous: walk up to a station kiosk or a dodgy mission giver, type or speak a line, and get an in-character reply spoken aloud with an on-screen subtitle. Third entry in the [the-experience-machine-ii](https://github.com/MIHLab-Org/the-experience-machine-ii) (Skyrim) / [the-experience-machine-fo4](https://github.com/MIHLab-Org/the-experience-machine-fo4) (Fallout 4) family — built differently, though: no Creation Kit or Papyrus here. It rides on Frontier's own sanctioned Journal file and the EDMC plugin ecosystem, the same approach used by [COVAS:NEXT](https://www.nexusmods.com/elitedangerous/mods/8).

See [BUILD.md](./BUILD.md) for the full step-by-step build doc, and the [Issues](../../issues) tab for the task breakdown from setup through stretch goals.
